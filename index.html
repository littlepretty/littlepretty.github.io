Setup Octopress

May 14th, 2014

Clone Octopress from Github

git clone git://github.com/imathis/octopress.git octopress cd octopress

Install Dependencies

sudo gem install bundler sudo gem install RedCloth rbenv rehash # If you use rbenv, rehash to be able to run the bundle command bundle install rake install

Deploy to Github

rake setup_github_pages rake generate rake deploy

Sync Source of Blog

cd _deploy git commit -m "commit message"

Blog Basics

Creates source/_posts/2011-07-03-zombie-ninjas-attack-a-survivors-retrospective.markdown

rake new_post["Zombie Ninjas Attack: A survivor's retrospective"] creates /source/super-awesome/index.markdown rake new_page[super-awesome] creates /source/super-awesome/page.html rake new_page[super-awesome/page.html]
